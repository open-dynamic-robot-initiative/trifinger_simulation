The SimFinger Class¶

The SimFinger class provides a simulation environment for the different finger robots: these include the TriFingerEdu, our reproducible open-source robot platform; the TriFingerPro, and the TriFingerOne.

The interface for controlling the robot in the simulation is equivalent to the one of robot_interfaces::RobotFrontend that is used for the real robots. Note, however, that there are some differences in behaviour between simulation and real robot, see Simulation vs Real Robot.

Class TriFingerPlatform is a wrapper around the SimFinger class. .

API Documentation¶

class trifinger_simulation.SimFinger(finger_type, time_step=0.004, enable_visualization=False)[source]¶

Provides simulation for each of the different finger robots: the trifingerone, trifingeredu, and trifingerpro, as well as complete API support for each of their single fingers (except for the trifingerpro). This simulation is based on PyBullet, the official Python wrapper around the Bullet-C API.
